# Installation

Clone repository
```
> git clone https://github.com/ruschakartur2/drf_test_task
```

Install dependencies

```
> pip install -r requirements.txt
```

Make migrations
```
> python manage.py makemigrations
```
Migrate
```
> python manage.py migrate
```
Start project
```
> python manage.py runserver
```
